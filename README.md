# Business Sales Performance Analytics

## Objective
Analyze business sales data to identify revenue trends, top products, and regional performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI

## Analysis Performed
- Category-wise sales analysis
- Region-wise sales analysis
- Top product analysis
- Monthly sales trend analysis

## Key Insights
- Technology category generated highest revenue
- West region showed strongest sales performance
- Phones and Chairs were top-selling products

## Recommendations
- Increase investment in Technology products
- Improve sales strategy in weaker regions
- Focus on seasonal promotions
